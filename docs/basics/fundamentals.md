## Variables(変数) & Constant(定数)

### 変数

```javascript
let userName = 'Max';
userName = 'Bob';
```

#### 命名

- camelCase での命名が基本
- 文字列と数字のみ使っていい
- $からはじめてもいい
- \_(アンダースコア)を使ってもいい

```javascript
let userName;
let ageGroup5;
let $kindOfSpecial;
let _internalValue;
```

### 定数

変数と違って一度定義したら変更できない

```javascript
const totalUsers = 15;
```

## 演算子

| 演算子 | 説明   |
| ------ | ------ |
| +      | 足し算 |
| -      | 引き算 |
| \*     | 掛け算 |
| /      | 割り算 |
| %      | 余り   |
| \*\*   | 累乗   |
| =      | 代入   |
